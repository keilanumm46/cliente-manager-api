
# Projeto de Cadastro de Clientes

Projeto simples que criei para praticar desenvolvimento de APIs com Spring Boot.

## Tecnologias
- Java
- Spring Boot
- JPA
- H2

## Como rodar

mvn spring-boot:run

## Endpoints

POST /clientes  
GET /clientes  
PUT /clientes/{id}  
DELETE /clientes/{id}

## Observações
- Não permite email duplicado
- Campos nome e email são obrigatórios
